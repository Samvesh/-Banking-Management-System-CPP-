# -Banking-Management-System-CPP-
A console-based banking management system built using Object-Oriented Programming concepts in C++. This project simulates core banking operations such as account creation, deposits, withdrawals, fund transfers, fixed deposits, EMI loans, transaction history, and file-based data persistence.
Features
Create Saving and Current accounts
Deposit and withdraw money
Transfer funds between accounts
Apply yearly interest and maintenance fees
Open Fixed Deposits (FD)
Create and manage EMI loans
Track transaction history
Search accounts by holder name
Persistent storage using text files
Menu-driven console interface
Concepts Used
Object-Oriented Programming (Classes & Objects)
File Handling
STL (vector, map)
Exception Handling
Time & Date utilities
Data Encoding/Decoding
Technologies
Language: C++
Compiler: G++
Standard Library (STL)
